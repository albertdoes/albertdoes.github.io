Elster / Palette

An Algorithmic color palette generator

Elster / Palette is a tool to help developer and designers generate color palette algorithmically using cubic Bezier curves and easing handles on the HSL space. So, it's easy for anyone who is not a stranger to cubic Bezier easing.


Features


1. Generous Easing

Elster / Palette offers a wide range of easing handles. This gives the flexibility for the users in generating colors.

2. Built-in Color Converter

Color generation is done only in HSL space. But users can still convert the colors into RGB or HEX or just stick to HSL.

3. Export to JSON

Elster / Palette makes using the colors in the user's preferred project environment easy by letting the user export the codes into JSON format.
